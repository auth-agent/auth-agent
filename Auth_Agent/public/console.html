Quick Start

Get started with Auth-Agent in 5 minutes

For Websites (OAuth Clients)

// 1. Create OAuth client above // 2. Install SDK npm install @auth-agent/web // 3. Initialize OAuth flow with PKCE import { AuthAgent } from '@auth-agent/web' const auth = new AuthAgent({ clientId: 'your_client_id', redirectUri: 'https://yoursite.com/callback' }) // Start OAuth flow (PKCE generated automatically) auth.login() // Handle callback const tokens = await auth.handleCallback(window.location.search)

For AI Agents

# 1. Register AI agent above # 2. Install SDK pip install auth-agent # 3. Authenticate with SDK from auth_agent import AgentSDK sdk = AgentSDK( agent_id="your_agent_id", agent_secret="your_secret" ) # Poll and authenticate auth_url = sdk.get_authorization_url( redirect_uri="https://website.com/callback", scope="openid profile email" ) # Open auth_url in browser, SDK polls automatically success = sdk.poll_and_authenticate(verify_ctx_id, timeout=30)

API Endpoints

Endpoint Method Description
/authorize GET OAuth 2.1 authorization endpoint (requires code_challenge)
/token POST Exchange authorization code for access token
/userinfo GET Get authenticated agent information
/verify_challenge GET Agent SDK polls for challenge
/verify_confirm POST Agent confirms authentication
